# Avatars

Coloca aquí las imágenes base de referencia de cada personaje (fondo neutro), con el
nombre que apunta `referenceImagePath` en `config.json`. Por ejemplo:

- `Avatars/elena.png`
- `Avatars/marcos.png`

Estas imágenes son la referencia de consistencia del avatar y se animan con HeyGen
(talking photo / Avatar IV). Más adelante se construirá una mini-colección de avatares
base reutilizables.
